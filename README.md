# Manasi's Virtual Creations — Portfolio

**Live site:**  https://manasi210.github.io/manasi-portfolio/

XR / VR developer portfolio built with a custom dc-runtime component framework. Showcases immersive projects in Unity and Unreal Engine — from full-body VR embodiment to AI-driven NPCs and mixed reality training apps.

## Stack

- Custom dc-runtime (React 18-based template engine)
- Vanilla CSS with CSS custom properties / design tokens
- Lucide icons
- No build step — open `Portfolio.dc.html` directly in a browser

## Projects featured

- Empathy Machine for Hard to Hear (VR / Unity)
- Vault Raiders: Save Verdantia (VR / Unity)
- Harry Potter Magical Mini-Golf (VR / Unity)
- Gambling Room (AI NPCs / Unity)
- Climate Change Awareness (VR / Unity)
- Modern Office Interiors (3D / Unreal Engine)
- Cafe Design (3D / Unity)
- Unreal Animation (3D / Unreal Engine)
- Festo Mixed Reality Experience (MR / Unity)

## Deployment

Hosted on GitHub Pages via the workflow in `.github/workflows/deploy.yml`. Pushes to `main` auto-deploy.

To enable: go to **Settings → Pages → Source** and set it to **GitHub Actions**.
